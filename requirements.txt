# Sneh App Requirements

## Backend
Node.js (v18 or higher)
npm (v9 or higher)

### Dependencies
@azure/openai^1.0.0-beta.12
cors^2.8.5
dotenv^16.3.1
express^4.18.2
ws^8.18.3

## Mobile App
Expo CLI (latest)
React Native (0.72+)

## Setup Instructions
1. **Backend**:
   - Navigate to `backend/`
   - Run `npm install`
   - Create a `.env` file with your Azure OpenAI keys:
     ```
     AZURE_OPENAI_ENDPOINT=...
     AZURE_OPENAI_KEY=...
     ```
   - Run `npm start`

2. **Mobile App**:
   - Navigate to `mobile-app/`
   - Run `npm install`
   - Run `npx expo start`
   - Scan the QR code with your phone (Expo Go app)

## Notes
- Ensure your phone and computer are on the same Wi-Fi network.
- Update `mobile-app/services/api.js` with your computer's local IP address if needed.
